## ✅ FASE 2: UNITY BUILDS (2-3 horas)

### **Para Cada Jogo:**
```
[ ] 1. Abrir projeto Unity
[ ] 2. File → Build Settings
[ ] 3. Switch Platform: Android
[ ] 4. Player Settings:
    - Package Name: com.afonso.fxtrade.[jogo]
    - Version: 1.0.0
    - Minimum API: Android 8.0
    - Scripting: IL2CPP
    - Architecture: ARM64
[ ] 5. Build APK
[ ] 6. Testar no dispositivo
[ ] 7. Assinar APK (keystore)
```

**Repetir 15 vezes = ~3 horas**

---

## ✅ FASE 3: PLAY STORE SETUP (2-3 horas)

### **Para Cada Jogo:**
```
[ ] 1. Google Play Console → Create App
[ ] 2. Preencher:
    - Nome: [Nome do Jogo]
    - idioma: English (US)
    - App ou Game: Game
    - Free ou Paid: Free
[ ] 3. Store Listing:
    - Title: [Nome do Jogo]
    - Short Description: [80 chars]
    - Full Description: [4000 chars]
    - Icon: Upload 1024x1024
    - Feature Graphic: Upload 1024x500
    - Screenshots: Upload 6+
    - Promo Video: Upload (opcional)
[ ] 4. Content Rating:
    - Preencher questionnaire
    - Obter rating (Everyone/Teen)
[ ] 5. Privacy Policy:
    - Upload PDF ou URL
[ ] 6. Pricing & Distribution:
    - Price: Free
    - Countries: All
    - Age restrictions: None
[ ] 7. Submit for Review
```

**Repetir 15 vezes = ~3 horas**

---

## ✅ FASE 4: ADMOB INTEGRATION (1-2 horas)

### **Para Cada Jogo:**
```
[ ] 1. AdMob → Create App

[ ] 3. Copiar IDs
[ ] 4. Atualizar Unity (AdMobManager.cs)
[ ] 5. Re-build APK
[ ] 6. Testar ads
[ ] 7. Upload nova versão
```

**Repetir 15 vezes = ~2 horas**
