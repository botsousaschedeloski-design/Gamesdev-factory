# 🎨 Canva Quick Start Guide

**Passo a Passo para Criar Assets em 3.5 Horas**

---

## 🚀 COMEÇAR AGORA:

### **Passo 1: Criar Conta Canva (2 min)**
```
1. Acesse: https://www.canva.com
2. Click: "Sign up" (grátis)
3. Login com Google/Facebook
4. ✅ Conta criada!
```

---

## 📱 ÍCONES (45 minutos):

### **Template Base:**
```
1. Acesse: https://www.canva.com/templates/logos/gaming/
2. Escolha template similar ao seu jogo
3. Click no template
4. Customize:
   - Cores (use paleta por país)
   - Elementos (cartas, dados, etc.)
   - Texto (remover se tiver)
5. Download:
   - Formato: PNG
   - Tamanho: 512x512
   - Fundo: Transparente (se necessário)
6. Repetir para 1024x1024 (redimensionar)
```

### **Tempo por Ícone: 3 minutos**
```
15 jogos × 3 min = 45 minutos
```

---

## 📸 SCREENSHOTS (2 horas):

### **Preparação (Unity):**
```
1. Abra Unity com cada jogo
2. Tire 8 screenshots de gameplay
3. Salve como PNG (1080x1920)
4. Repita para 15 jogos
```

### **Edição (Canva):**
```
1. Acesse: https://www.canva.com/templates/social-media-posts/phone-screens/
2. Escolha template "App screenshot"
3. Upload da screenshot do Unity
4. Adicionar overlay text:
   - "Jogue Grátis!"
   - "100% Offline"
5. Download: PNG, 1080x1920
6. Repetir 8x por jogo
```

### **Tempo por Screenshot: 8 minutos**
```
15 jogos × 8 screenshots × 8 min = 16 horas (se fazer tudo)

OU

Atalho: 3 screenshots por jogo (suficiente)
15 jogos × 3 screenshots × 8 min = 6 horas

OU

Super Atalho: Usar screenshots diretas do Unity (sem edição)
15 jogos × 8 screenshots = 2 horas (apenas capturar)
```

---

## 🎨 FEATURE GRAPHICS (45 minutos):

### **Template Base:**
```
1. Acesse: https://www.canva.com/templates/banners/gaming/
2. Escolha template 1024x500
3. Customize:
   - Logo do jogo
   - 2-3 screenshots
   - Texto: "Jogue Grátis!"
4. Download: PNG, 1024x500
5. Repetir para 15 jogos
```

### **Tempo por Feature: 3 minutos**
```
15 jogos × 3 min = 45 minutos
```

---

## ⚡ MODO SUPER RÁPIDO (1.5 horas):

### **Se quiser lançar RÁPIDO:**

#### **Ícones (30 min):**
```
1. Usar templates prontos do Canva
2. Só mudar cores
3. 2 min por ícone × 15 = 30 min
```

#### **Screenshots (1h):**
```
1. Capturar direto do Unity (sem edição)
2. 4 min por jogo × 15 = 1h
3. Mínimo 3 screenshots por jogo
```

#### **Feature Graphics (15 min):**
```
1. Usar template único
2. Só mudar logo/texto
3. 1 min por jogo × 15 = 15 min
```

**TOTAL: 1.5 horas (mínimo viável)**
