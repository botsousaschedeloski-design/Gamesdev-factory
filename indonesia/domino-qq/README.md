## 🚀 Como Usar no Unity:

### **Passo 1: Importar Scripts**

```
1. Copie pasta Scripts/ para Assets/Scripts/
2. Unity vai compilar automaticamente
```

### **Passo 2: Criar Prefabs**

```
1. Crie prefab para DominoPiece
2. Adicione SpriteRenderer
3. Configure sprites das peças
```

### **Passo 3: Configurar Cena**

```
1. Crie GameObject "GameManager"
2. Adicione componente DominoGame
3. Referencie prefabs
4. Configure board, hands, UI
```

### **Passo 4: Testar**

```
1. Play no Unity
2. Arraste peças para jogar
3. Bots jogam automaticamente
```
